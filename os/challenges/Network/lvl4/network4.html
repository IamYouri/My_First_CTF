TCP SYN FLOOD

Pour ce challenge on aura besoin d'un serveur

Démarrez un serveur python dans votre terminal

Python3 -m http.server 1234

Désactivez la protection contre les attaques SYN Flooding

sudo sysctl -w net.ipv4.tcp_syncookies=0

Modifiez la taille de la file d'attente

sudo sysctl net.ipv4.tcp_max_syn_backlog=5

Attaquez le serveur

La commande pour faire cette attaque est la clé d'accès

N'oubliez pas d'installer netwox pour ce challenge
